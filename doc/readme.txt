Overview
========
The 3-phase brushless DC motor (BLDC) control reference application
is  intended to provide the example for 3-phase BLDC motor control solutions.
The reference design uses a communitation control mechanism.

More information
================
- Release note is in SDK install folder\docs\MC\Release_notes.pdf.
- Application page: http://www.nxp.com/motorcontrol_pmsm.

Running the demo
================
- Follow application note "BLDC Motor Control Demo on LPC553x/LPC55S3x ".
